Fake News Classification

Machine learning system to classify news articles as Fake or Real

NLP · TF-IDF · Supervised Learning · Python

View Code

Overview

This project focuses on detecting fake news using machine learning and natural language processing techniques. With the rapid spread of misinformation through digital platforms, automated classification systems are essential for identifying unreliable content.

Problem

Fake news spreads rapidly and is difficult to detect manually. There is a need for a system that can automatically analyze news content and classify it as real or fake with high accuracy.

Solution

Built a supervised machine learning model that processes textual data, extracts features, and classifies news articles using NLP techniques and classification algorithms.

Workflow

  • Data cleaning — removing punctuation, stopwords, and normalization
  • Tokenization and text processing
  • Feature extraction using TF-IDF vectorization
  • Model training and testing on labeled dataset
  • Performance evaluation using metrics

Model

Implemented supervised classification models including Logistic Regression and Passive-Aggressive Classifier. TF-IDF vectorization was used to convert textual data into numerical features for training.

Tech Stack

Language: Python

Libraries: Pandas, NumPy, Scikit-learn, Matplotlib

Results

The model achieved an accuracy of 93.58% on test data, with a precision of 0.93 and recall of 0.94.

  • Accuracy Score: 93.58%
  • Precision: 0.93
  • Recall: 0.94
  • Evaluation via Confusion Matrix

Dataset

The dataset consists of labeled news articles categorized as Fake or Real, including features such as title, text content, and labels used for training the classification model.
