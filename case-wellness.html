💪 Wellness Personas of SNU

ML-powered lifestyle persona discovery app using Streamlit & KMeans Clustering

Python · scikit-learn · Streamlit · Matplotlib

Live Demo View Code

Project Overview

Wellness Personas of SNU is an interactive web application that identifies a user's lifestyle persona based on daily habits and preferences. It combines machine learning with a human-centered design approach to help students better understand their wellness patterns.

Problem

Students often lack awareness about their lifestyle habits and how those habits affect their overall wellness. There was no engaging, personalized way to reflect on daily behaviors.

Solution

Designed an interactive system that collects user inputs and uses KMeans clustering to map users into meaningful lifestyle personas, providing personalized insights and motivation.

Personas (Clusters)

  • 🍕 Social Foodie – Loves cafes and social dining
  • 🥗 Trendy Eater – Follows food trends
  • 🍰 Sweet Lover – Dessert enthusiast
  • ☕ Classic Soul – Calm and balanced lifestyle
  • 🎨 Creative Hobbyist – Artistic and expressive
  • 📚 Chill Introvert – Enjoys solitude and books
  • 🧘 Balanced Persona – Maintains harmony
  • 💪 Active Extrovert – Fitness and social life
  • 🌿 Minimalist – Simple and mindful living
  • 🔥 Passionate Creator – Driven and focused

Key Features

  • Interactive lifestyle input sliders
  • Dynamic persona visualization (radar chart)
  • Dark-themed UI design
  • Personalized quotes and feedback
  • Fast and responsive Streamlit interface

Tech Stack

Frontend: Streamlit (Custom UI)

Backend: Python

ML: KMeans Clustering (Scikit-learn)

Visualization: Matplotlib

Libraries: NumPy, Pandas, Joblib

Dataset

The dataset contains lifestyle-related inputs such as food habits, hobbies, and social activity used for training the ML model.
