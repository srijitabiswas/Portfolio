ZeroShield AI

AI-powered cybersecurity platform for real-time threat detection

Full-stack · React · Node.js · TypeScript · Tailwind CSS

View Code

Overview

ZeroShield AI is a full-stack cybersecurity platform designed to simulate a real-world Security Operations Center (SOC). It provides real-time monitoring, AI-based anomaly detection, attack simulation, and automated response handling through a centralized dashboard.

Problem

Traditional cybersecurity systems are often complex, fragmented, and difficult to interpret. There is a need for a unified platform that simplifies threat monitoring, detection, and response in a clear and actionable way.

Solution

Designed a centralized SOC dashboard that integrates monitoring, detection, simulation, and response into a single interface, enabling users to understand and react to threats efficiently.

Key Features

  • Real-time risk score and threat monitoring
  • AI-based anomaly detection with confidence metrics
  • Attack simulation engine (API injection, DDoS, etc.)
  • Automated response actions (block IP, isolate service)
  • Threat intelligence feed with severity classification
  • Real-time logs and activity tracking
  • Notification system with alerts and updates

Dashboard Design

The interface is designed to resemble a modern SOC dashboard, focusing on clarity, hierarchy, and real-time visibility. Key metrics like risk score, active threats, and system activity are prioritized to support quick decision-making.

System Architecture

Built using a modular full-stack architecture with a React-based frontend and Node.js backend. The system includes a simulation engine, AI logic layer, and centralized API communication.

Tech Stack

Frontend: React (Vite), TypeScript, Tailwind CSS, Radix UI

Backend: Node.js

Visualization: Recharts

Architecture: Modular API services + simulation engine

Highlight: Attack Simulation Engine

Implemented a simulation engine to mimic real-world cyberattacks such as DDoS spikes, API injection, and lateral movement, allowing users to visualize and understand threats dynamically.
